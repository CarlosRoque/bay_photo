- (EnumInstance) initialize(input)

This method is part of a private API. You should avoid using this method if possible, as it may be removed or be changed in the future.

Create a new EnumInstance with #value set to the correspondance of the input. If the input is a Symbol, the #value is set to the Integer value of the Enum definition. If the input is an Integer, it's the reverse.

    the Symbol or Integer to convert

def initialize(input)
  @value = case input
           when Integer
             sym = args[input - start_at]
             fail "Enum out of bounds: #{input}" if sym.nil?
             sym
           when Symbol
             int = args.index(input)
             fail "Enum out of bounds: #{input}" if int.nil?
             int + start_at
           else
             fail "Invalid Enum input: #{input}"
           end
end
